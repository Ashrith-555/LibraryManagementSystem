# Library Management System

## Project Overview
Library Management System is a desktop-based application developed to automate and simplify daily library operations such as managing books, members, and loan transactions.

## Features
- Secure login system
- Book management (Add, View, Update, Delete, Search)
- Member management (Add, View, Update, Delete)
- Borrow book functionality
- Return book functionality
- Automatic book availability status update
- User-friendly graphical interface

## Technologies Used
- Python
- Tkinter
- SQLite3
- Pillow (for background images)

## Project Structure
- backend → Database connection and business logic
- ui → User interface screens
- images → Background images used in modules
- database → SQLite database file
- main.py → Main application entry point
